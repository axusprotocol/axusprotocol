AXUS Protocol: Whitepaper

1. Summary

AXUS Protocol is a decentralized smart-collateral platform built on the TON blockchain. It allows users to deposit almost any token ‚Äî even low-liquidity or airdropped assets ‚Äî and receive AXP tokens in return. AXUS revives forgotten or idle assets and turns them into usable liquidity.

2. The Problem

Millions of crypto users hold tokens that are illiquid, unused, or come from failed projects. These tokens are often ignored by DeFi platforms and simply remain stuck in wallets without any value or purpose.

3. The Solution: AXUS

AXUS accepts up to 3 tokens per request as collateral. The protocol uses oracles to evaluate their combined market value and provides 60% of that value in AXP tokens ‚Äî instantly, via smart contract. No middlemen. No manual review.

4. How It Works

  1. User submits 1 to 3 tokens
  2. AXUS calculates their value via decentralized oracles
  3. User receives 60% of the total value in AXP
  4. Loan terms: 10, 20, 30, or 60 days with fixed interest
  5. If not repaid on time, collateral is retained by the protocol

5. Tokenomics (AXP)

  • Total supply: TBD
  • AXP is minted when loans are issued
  • AXP is burned when loans are repaid
  • Used for staking, governance, and internal utilities

6. Features

  • Accepts any token that is at least 3 months old
  • No KYC required
  • Fully on-chain and autonomous
  • Fixed request fee: 0.01 TON

7. Roadmap

  • NFT collateral support
  • On-chain reputation scoring
  • AXUS mobile app
  • Cross-chain support

8. Governance

AXUS starts as a fully autonomous system. DAO functionality (such as voting and proposal submission) may be introduced later based on stability and ecosystem demand.

9. Conclusion

AXUS Protocol creates a new way to unlock liquidity from forgotten tokens. Through transparent automation and smart contracts, users can turn passive assets into active capital ‚Äî instantly, and without banks or brokers.
